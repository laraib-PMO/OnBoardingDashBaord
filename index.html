ProductPhase 0 active

Attimo is not a task manager and not a chatbot. It is a governed, evidence-native coordination layer. Every output cites its source. Every action is approval-gated. Every decision is logged immutably.

8 core user stories
US1
Recover context
Find the right info from vague or partial memory
US2
Check accepted state
What is the current approved version?
US3
Track obligations
What do we owe, to whom, by when?
US4
Generate report
Site audit report from field evidence
US5
Capture field signal
Photo or voice from site — auto-structured
US6
Review and approve
Approve obligation closure or state changes
US7
Monitor deadlines
Alerts before obligations go overdue
US8
Connect channels
Link email, drive, messaging for auto-ingestion
Non-negotiable rules
State engine is always a deterministic rules engine — never AI
Deviation detection is always a graph query — never AI
Every external or contractual action requires human approval before execution
Every derived output carries provenance — no silent AI decisions, ever

MarketingAEC · Turkey + EU
The problem
Construction projects generate thousands of signals that are never captured or linked to decisions. When disputes or audits happen, nobody has a single source of truth.
The solution
A coordination layer on top of existing tools without forcing migration. Connects fragmented signals into governed, traceable records.
What it is not
Not a task managerNot a chatbotNot a document managerNot another PM toolNot a migration project
5 value propositions
Find any project context in seconds — with proof of why it was returned
Always know the current approved version of any document or decision
Capture field signals automatically — photo, voice, text — no office software needed
Track every obligation with its source clause attached through to closure
Export audit-ready evidence packages for disputes, handovers, and claims instantly
