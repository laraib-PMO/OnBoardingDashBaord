Clarified HITL concept and augmented tools grid with additional AI platforms Clarified HITL concept and augmented tools grid with additional AI platforms HITL = Human-in-the-Loop — not HILT. It's a term from AI system design. In Attimo, every time an AI agent produces an output that could have a real consequence (creating an obligation, accepting a document version, sending a report, closing a task), the system pauses and waits for a human to review and approve before anything is executed. That pause point is called a gate. There are 7 defined gates (H1–H7) in Attimo's architecture: H1 — Obligation confirmation (AI extracts, human confirms) H2 — State acceptance (human accepts a document revision) H3 — Report expert review (human edits the AI draft) H4 — Report final approval (human signs off before export) H5 — Obligation closure (human closes with evidence) H6 — User role changes (owner approves) H7 — Obligation deletion (owner approves) The core principle: the AI drafts, proposes, and structures — but a human always decides before anything binding happens.
